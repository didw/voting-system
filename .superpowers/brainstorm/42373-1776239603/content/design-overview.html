B안 상용화 개선 전체 설계. 각 섹션을 확인 후 피드백 주세요.

섹션 1 · DB 마이그레이션

devices 테이블에 컬럼 1개 추가

ALTER TABLE devices ADD COLUMN last_voted_at TIMESTAMP NULL COMMENT '마지막 투표 버튼 누른 시각';
  • last_seen_at → 전원을 켠 시각 (mac: 메시지 수신 시 갱신)
  • last_voted_at → 투표 버튼을 누른 시각 (vote: 메시지 수신 시 갱신)

섹션 4 · 행운권 추첨 수정

현재 (버그): 서버가 뽑은 번호와 바퀴 애니메이션 번호가 다름

수정 방향:

  1. 추첨 버튼 클릭 → 서버 API 먼저 호출하여 당첨 번호 확정
  2. 확정된 번호를 LotteryWheel에 targetNumber prop으로 전달
  3. 바퀴가 그 번호로 정확히 랜딩하는 애니메이션 실행
  4. 로딩 중에는 "추첨 중..." 스피너 표시
